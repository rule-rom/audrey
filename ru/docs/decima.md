## Концепция

Ламповая Децима — это **аппаратный сопроцессор** для игр, который использует 
аналоговую природу вакуумных ламп для эвристического принятия решений.

### Зачем лампы в играх?

| Цифра | Аналог |
|-------|--------|
| Дискретные вычисления | Непрерывная физика |
| Детерминированные алгоритмы | Аналоговая эвристика |
| Микросекундные задержки | Мгновенный отклик |
| Холодный расчёт | Тёплая непредсказуемость |

## Архитектура

### Ядро: 16 баллонов

```
┌─────────────────────────────────────────────────────┐
│              Ламповая Децима (16 баллонов)          │
├─────────────────────────────────────────────────────┤
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                │
│  │ L1  │  │ L2  │  │ L3  │  │ L4  │  → Зрительный  │
│  └─────┘  └─────┘  └─────┘  └─────┘    канал       │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                │
│  │ L5  │  │ L6  │  │ L7  │  │ L8  │  → Двигатель   │
│  └─────┘  └─────┘  └─────┘  └─────┘    прицела    │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                │
│  │ L9  │  │ L10 │  │ L11 │  │ L12 │  → Тактика     │
│  └─────┘  └─────┘  └─────┘  └─────┘    решения     │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                │
│  │ L13 │  │ L14 │  │ L15 │  │ L16 │  → Выход       │
│  └─────┘  └─────┘  └─────┘  └─────┘    триггеры    │
└─────────────────────────────────────────────────────┘
```

### Функциональные блоки

| Блок | Лампы | Назначение |
|------|-------|------------|
| **Зрительный канал** | L1–L4 | Обработка видеопотока, выделение целей |
| **Двигатель прицела** | L5–L8 | Аналоговое ведение прицела к цели |
| **Тактика** | L9–L12 | Принятие решений (атака/укрытие) |
| **Выход** | L13–L16 | Триггеры: выстрел, движение, уклонение |

## Принципы работы

### Нейроморфная аналогия

```
Синапс (биология) → Ламповый каскад (аналог)
Нейрон (биология) → Суммирующая сетка (аналог)
Потенциал действия → Анодный импульс
```

### Обработка сигнала

1. **Вход**: Видеосигнал с захвата (HDMI/USB)
2. **Аналоговая обработка**: Лампы усиливают «интересные» участки
3. **Пороговое срабатывание**: При превышении — триггер
4. **Выход**: USB/HID → игра (выстрел, движение)

| **Потребление** | ~150 Вт |

## Режимы работы

### 🎯 Auto-Aim

Аналоговое ведение прицела по контрастным объектам.

```
Видео → Контраст → Усилитель → Отклонение прицела
```

### 🔥 Auto-Fire

Пороговое срабатывание при попадании цели в сектор.

```
Сектор → Компаратор (лампа) → Триггер → Выстрел
```

### 🧠 Тактика

Простые эвристики на аналоговой логике:

- **Укрытие**: При низком HP — отход
- **Атака**: При преимуществе — вперёд
- **Патруль**: Случайное блуждание (ламповый шум)

Ламповая Децима использует принципы корневой Decima8:
