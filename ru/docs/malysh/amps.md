## Спецификации

| Параметр | Значение |
|----------|----------|
| **Тип схемы** | Двухтактный Push-Pull класса А/АВ |
| **Выходные лампы** | Г-807 (2 шт.) |
| **Драйвер/Фазоинвертор** | 6Н23П-ЕВ (2 шт.) |
| **Мощность** | 2×15 Вт (на нагрузке 4-8 Ом) |
| **Частотный диапазон** | 15 Гц – 40 кГц (±1 дБ) |
| **КНИ** | < 1% при номинальной мощности |
| **Входы** | RCA (стерео) |
| **Регулятор громкости** | ALPS (моторизованный) |

---

## Архитектура сигнала

```
Вход RCA → Регулятор ALPS → 6Н23П-ЕВ (драйвер) → 
Фазоинвертор → Г-807 (выходной каскад) → ТВЗ → Акустика
```

### Каскады

1. **Входной каскад** — моторизованный потенциометр ALPS
2. **Драйвер** — 6Н23П-ЕВ, усиление напряжения
3. **Фазоинвертор** — схема с раздельной нагрузкой
4. **Выходной каскад** — Г-807 в двухтактном включении
5. **Трансформаторы** — ТВЗ на ОСМ-сердечниках

---

## Режимы работы

| Режим | Ток покоя | Класс | Звук |
|-------|-----------|-------|------|
| **A** | 60 мА | Чистый класс A | Мягкий, детальный |
| **AB** | 45 мА | Класс AB | Динамичный, мощный |

Режим выбирается через tube контроллер или задается фиксированно в БП.

## Измерения и калибровка

### ВАХ-метр

Встроенная система автоматически снимает характеристики ламп:

- **Ia (ток анода)** — в зависимости от напряжения
- **Vg (напряжение сетки)** — кривые смещения
- **Эмиссия** — остаточный ресурс катода

### Auto-Bias

Tube контроллер удерживает ток покоя с точностью до **0,1 мА**:

- Автоматическая подстройка при прогреве
- Компенсация старения ламп
- Уведомления о деградации

## Подключение

```
[Источник] → RCA → [Малыш] → Акустика
                    ↓
              USB/Ethernet → ПК (Lisp-агент)
```

### Интерфейсы

| Порт | Назначение |
|------|------------|
| **RCA** | Стерео вход (линейный уровень) |
| **USB** | Отладка, прошивка, снятие ВАХ |
| **Ethernet** (серия) | Сеть, удалённый мониторинг |
