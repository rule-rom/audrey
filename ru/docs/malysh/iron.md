## Выходные трансформаторы (ТВЗ)

### Серия: ОСМ-0.16 (заводское исполнение)

| Параметр | Значение |
|----------|----------|
| **Базовый сердечник** | ОСМ-0.16 (разрезной витой сердечник, тип ШЛ 32×40) |
| **Материал стали** | Холоднокатаная электротехническая сталь (марки 3407 или 3408) |
| **Конструктив** | Индивидуальный расчет под Push-Pull, стальной экран («кастрюля») |
| **Технология** | Секционированная намотка (3П + 2В), вакуумная пропитка лаком |
| **Диапазон частот** | 15 Гц — 45 000 Гц (при неравномерности -1 дБ) |
| **Особенности** | Индивидуальный паспорт ВАХ для каждой пары трансформаторов |
| **Производство** | Заводская намотка, контроль качества |

## Намоточные данные (для самостоятельного изготовления)

### ТВЗ на ОСМ-0.16

```
Первичная обмотка:
  - Провод: ПЭТВ-2 0.25 мм
  - Витков: 5000
  - Секции: 3П+2В (чередование)
  - Сопротивление: ~180 Ом

Вторичная обмотка:
  - Провод: ПЭТВ-2 0.8 мм
  - Витков: 120 (отвод на 4 Ом), 170 (отвод на 8 Ом)
  - Секции: 2 параллельно

Изоляция:
  - Межсекционная: лакоткань 0.1 мм
  - Межобмоточная: 3 слоя бумаги
  - Пропитка: парафин + воск
```

### Дроссель ОСМ-0.063

```
Обмотка:
  - Провод: ПЭТВ-2 0.35 мм
  - Витков: 2000
  - Зазор: 0.1 мм (текстолит)
  - Индуктивность: ~8 Гн при 100 мА
```

## Материалы

### Сердечники

| Тип | Материал | Проницаемость |
|-----|----------|---------------|
| **ОСМ** | Сталь Э310 | μ ~ 2000 |

### Провода

- **ПЭТВ-2** — полиэфиримидная эмаль, класс нагревостойкости F (155°C)
- **Медь** — бескислородная, 99.99%

### Изоляция

- **Лакоткань** — 0.1 мм, класс F
- **Бумага** — конденсаторная, 0.02 мм
- **Пропитка** — парафин + пчелиный воск (1:1)
