## 规格

| 参数 | 值 |
|------|-----|
| **电路类型** | 推挽式 A/AB 类 |
| **输出管** | G-807 (2 件) |
| **驱动/倒相** | 6N23P-EV (2 件) |
| **功率** | 2×15W (4-8 欧姆负载) |
| **频率范围** | 15 Hz – 40 kHz (±1 dB) |
| **总谐波失真** | < 1% (额定功率下) |
| **输入** | RCA (立体声) |
| **音量控制** | ALPS (电动) |

---

## 信号架构

```
RCA 输入 → ALPS 电位器 → 6N23P-EV (驱动) → 
倒相器 → G-807 (输出级) → TVZ → 扬声器
```

### 级

1. **输入级** — 电动 ALPS 电位器
2. **驱动级** — 6N23P-EV，电压增益
3. **倒相器** — 分离负载电路
4. **输出级** — G-807 推挽配置
5. **变压器** — OSM 铁芯 TVZ

---

## 工作模式

| 模式 | 静态电流 | 类别 | 声音 |
|------|----------|------|------|
| **A** | 60 mA | 纯 A 类 | 柔和，细腻 |
| **AB** | 45 mA | AB 类 | 动态，强劲 |

模式通过 tube 控制器选择或在电源中固定设置。

## 支持的电子管

得益于通用平台和 Decima 控制器，Малыш 支持输出电子管热插拔：

| 电子管 | 等效 | 声音特征 |
|--------|------|----------|
| **G-807** | — | 魅力和力量，Mалыш 标志性声音 |
| **6P3S** | 6L6 | 经典，温暖低音 |
| **EL34** | — | 英伦摇滚，激进中频 |
| **6P36S** | — | 低阻抗，低音控制 |
| **6P44S** | — | 细腻高音，快速响应 |
| **GU-50** | — | 坦克之声，最大功率 |

## 测量和校准

### 特性曲线仪

内置系统自动记录电子管特性：

- **Ia (阳极电流)** — 随电压变化
- **Vg (栅极电压)** — 偏置曲线
- **发射** — 阴极剩余寿命

### Auto-Bias

Tube 控制器以 **0.1 mA** 精度保持静态电流：

- 预热期间自动调整
- 补偿电子管老化
- 退化通知

## 连接

```
[音源] → RCA → [Малыш] → 扬声器
                    ↓
              USB/以太网 → PC (Lisp 代理)
```

| 端口 | 用途 |
|------|------|
| **RCA** | 立体声输入 (线路电平) |
| **USB** | 调试，固件，特性曲线 |
| **以太网** (量产) | 网络，远程监控 |
