## 输出变压器 (TVZ)

### 量产：OSM-0.16 (工厂制造)

| 参数 | 值 |
|------|------|
| **基础铁芯** | OSM-0.16 (切割卷绕铁芯，类型 ШЛ 32×40) |
| **钢材材料** | 冷轧电工钢 (牌号 3407 或 3408) |
| **结构** | 推挽式定制设计，钢屏蔽 ("锅") |
| **工艺** | 分段绕制 (3P + 2S)，真空漆浸渍 |
| **频率范围** | 15 Hz — 45,000 Hz (-1 dB 不均匀度) |
| **特点** | 每对变压器配备独立 Ia/Vg 特性曲线护照 |
| **生产** | 工厂绕制，质量控制 |

## 绕制数据 (DIY 参考)

### TVZ on OSM-0.16

```
初级绕组:
  - 线：PETV-2 0.25 mm
  - 匝数：5000
  - 分段：3P+2S (交替)
  - 电阻：~180 欧姆

次级绕组:
  - 线：PETV-2 0.8 mm
  - 匝数：120 (4 欧姆抽头)，170 (8 欧姆抽头)
  - 分段：2 并联

绝缘:
  - 段间：漆布 0.1 mm
  - 绕组间：3 层纸
  - 浸渍：石蜡 + 蜂蜡
```

### 扼流圈 OSM-0.063

```
绕组:
  - 线：PETV-2 0.35 mm
  - 匝数：2000
  - 间隙：0.1 mm (电木)
  - 电感：~8 H (100 mA 时)
```

## 材料

### 铁芯

| 类型 | 材料 | 磁导率 |
|------|------|--------|
| **OSM** | 钢 E310 | μ ~ 2000 |

### 导线

- **PETV-2** — 聚酰亚胺漆，耐热等级 F (155°C)
- **铜** — 无氧，99.99%

### 绝缘

- **漆布** — 0.1 mm，F 级
- **纸** — 电容级，0.02 mm
- **浸渍** — 石蜡 + 蜂蜡 (1:1)
